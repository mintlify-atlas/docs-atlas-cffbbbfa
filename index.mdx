    Quick start
  </h2>
  <p className="dark:text-gray-400 text-gray-600 mb-8">
    Get up and running with Upsonic in minutes
  </p>

  <Steps>
    <Step title="Install Upsonic">
      Install the package using pip or uv:

      ```bash
      pip install upsonic
      ```

      For additional features, install with extras:

      ```bash
      pip install "upsonic[ocr,storage,models]"
      ```
    </Step>

    <Step title="Set up your API key">
      Configure your AI provider credentials:

      ```bash
      export ANTHROPIC_API_KEY="your-api-key"
      # or
      export OPENAI_API_KEY="your-api-key"
      ```

      Upsonic supports OpenAI, Anthropic, Azure, AWS Bedrock, Google, and more.
    </Step>

    <Step title="Create your first agent">
      Create a simple agent and run a task:

      ```python
      from upsonic import Agent, Task

      agent = Agent(
          model="anthropic/claude-sonnet-4-5",
          name="Assistant"
      )

      task = Task(description="Explain quantum computing in simple terms")
      result = agent.do(task)
      print(result)
      ```

      Your agent is now ready to execute tasks!
    </Step>
  </Steps>

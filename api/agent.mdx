## Overview

A comprehensive, high-level AI Agent that integrates all framework components including model abstraction, tool handling, streaming execution, memory management, context management, caching, safety policies, and more.

## Class Definition

```python
from upsonic import Agent

agent = Agent(
    model="openai/gpt-4o",
    name="MyAgent",
    memory=memory,
    tools=[tool1, tool2]
)
```

## Core Methods

### do()

Execute a task synchronously and return the result.

```python
result = agent.do(task)
```

<ParamField path="task" type="Union[str, Task, List[Union[str, Task]]]" required>
  Task object, string description, or list of tasks to execute
</ParamField>

<ResponseField name="result" type="Any">
  The task result - typically a string, but can be a structured Pydantic model if response_format is specified
</ResponseField>

**Example:**

```python
from upsonic import Agent, Task

agent = Agent("openai/gpt-4o")
task = Task("What is 2 + 2?")
result = agent.do(task)
print(result)  # "4"
```

### do_async()

Execute a task asynchronously.

```python
result = await agent.do_async(task)
```

<ParamField path="task" type="Union[str, Task, List[Union[str, Task]]]" required>
  Task object, string description, or list of tasks
</ParamField>

<ParamField path="return_output" type="bool" default="False">
  If True, return full AgentRunOutput instead of just the content
</ParamField>

<ResponseField name="result" type="Union[Any, AgentRunOutput]">
  Task result or full AgentRunOutput if return_output=True
</ResponseField>

### stream()

Stream task execution synchronously with real-time output.

```python
for chunk in agent.stream(task):
    print(chunk, end="", flush=True)
```

<ParamField path="task" type="Union[str, Task]" required>
  Task object or string description
</ParamField>

<ParamField path="events" type="bool" default="True">
  If True, yield AgentStreamEvent objects; if False, yield text strings only
</ParamField>

<ResponseField name="iterator" type="Iterator[Union[str, AgentStreamEvent]]">
  Iterator yielding text chunks or event objects
</ResponseField>

### astream()

Stream task execution asynchronously.

```python
async for chunk in agent.astream(task):
    print(chunk, end="", flush=True)
```

<ParamField path="task" type="Union[str, Task]" required>
  Task object or string description
</ParamField>

<ParamField path="events" type="bool" default="True">
  If True, yield AgentStreamEvent objects; if False, yield text strings only
</ParamField>

<ResponseField name="async_iterator" type="AsyncIterator[Union[str, AgentStreamEvent]]">
  Async iterator yielding text chunks or event objects
</ResponseField>

### add_tools()

Dynamically add tools to the agent after initialization.

```python
agent.add_tools([tool1, tool2])
```

<ParamField path="tools" type="Union[Any, List[Any]]" required>
  Single tool or list of tools to add (functions, ToolKits, agents, etc.)
</ParamField>

### cancel_run()

Cancel a running task execution.

```python
agent.cancel_run(run_id="abc123")
```

<ParamField path="run_id" type="str" default="None">
  ID of the run to cancel. If None, cancels the current run.
</ParamField>

<ResponseField name="success" type="bool">
  True if the run was found and cancelled
</ResponseField>

## Usage Methods

### get_run_output()

Get the complete output from the last execution.

```python
output = agent.get_run_output()
```

<ResponseField name="output" type="AgentRunOutput">
  Complete run output with messages, usage, tool calls, etc.
</ResponseField>

### get_session_usage()

Get aggregated usage statistics for the current session.

```python
usage = agent.get_session_usage()
print(f"Total cost: ${usage.total_cost}")
print(f"Input tokens: {usage.total_input_tokens}")
```

<ResponseField name="usage" type="RunUsage">
  Aggregated usage metrics including tokens and costs
</ResponseField>

### get_cache_stats()

Get cache performance statistics.

```python
stats = agent.get_cache_stats()
```

<ResponseField name="stats" type="Dict[str, Any]">
  Cache statistics including hit rate and total entries
</ResponseField>

### clear_cache()

Clear the agent's session cache.

```python
agent.clear_cache()
```

## Advanced Examples

### Multi-Tool Agent

```python
from upsonic import Agent, Task

def calculate(a: int, b: int) -> int:
    """Add two numbers."""
    return a + b

def search_web(query: str) -> str:
    """Search the web."""
    return f"Results for {query}"

agent = Agent(
    model="openai/gpt-4o",
    name="Assistant",
    tools=[calculate, search_web],
    show_tool_calls=True
)

task = Task("What is 25 + 17, and search for Python tutorials")
result = agent.do(task)
```

### Agent with Memory

```python
from upsonic import Agent, Memory, InMemoryStorage

storage = InMemoryStorage()
memory = Memory(
    storage=storage,
    full_session_memory=True
)

agent = Agent(
    model="openai/gpt-4o",
    memory=memory
)

# First conversation
agent.do("My name is Alice")

# Agent remembers
result = agent.do("What's my name?")  # "Your name is Alice"
```

### Streaming with Events

```python
from upsonic import Agent, Task

agent = Agent("openai/gpt-4o")
task = Task("Write a short poem")

for event in agent.stream(task, events=True):
    if event.event_type == "text_delta":
        print(event.data, end="", flush=True)
    elif event.event_type == "tool_call":
        print(f"\nCalling tool: {event.data['name']}")
```

### Safety Policies

```python
from upsonic import Agent, Task
from upsonic.safety_engine import AdultContentPolicy

policy = AdultContentPolicy()

agent = Agent(
    model="openai/gpt-4o",
    user_policy=policy,  # Filter user inputs
    agent_policy=policy  # Filter agent outputs
)

task = Task("Tell me about AI ethics")
result = agent.do(task)
```

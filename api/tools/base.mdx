## Overview

The Upsonic tools system provides a comprehensive framework for creating and managing AI agent tools. This module contains the core base classes, interfaces, and tool management functionality.

## Tool Base Class

The `Tool` class is the central base class for all tools in the Upsonic framework.

### Tool

**Location:** `src/upsonic/tools/base.py:49`

```python
class Tool:
    """Central base class for all tools in the Upsonic framework."""
    
    def __init__(
        self,
        name: str,
        description: Optional[str] = None,
        schema: Optional[FunctionSchema] = None,
        metadata: Optional[ToolMetadata] = None,
        tool_id: Optional[str] = None,
    )
```

#### Purpose

- Standard tool interface (name, description, metadata)
- Metrics tracking for each tool instance
- Abstract execute method for tool logic

#### Parameters

- `name` (str): Tool name
- `description` (Optional[str]): Tool description
- `schema` (Optional[FunctionSchema]): Tool's input schema
- `metadata` (Optional[ToolMetadata]): Tool metadata
- `tool_id` (Optional[str]): Unique identifier (auto-generated if not provided)

#### Example Usage

```python
from upsonic.tools import Tool, ToolMetadata

class MyTool(Tool):
    def __init__(self):
        super().__init__(
            name="my_tool",
            description="Does something useful",
            metadata=ToolMetadata(name="my_tool", kind='function')
        )
    
    async def execute(self, **kwargs):
        # Tool logic here
        result = perform_operation(kwargs)
        return result

# Use the tool
```

## ToolKit Base Class

**Location:** `src/upsonic/tools/base.py:165`

```python
class ToolKit:
    """Base class for organized tool collections.
    
    Only @tool decorated methods are exposed as tools.
    """
```

### Example Usage

```python
from upsonic.tools import tool, ToolKit

class MyToolKit(ToolKit):
    @tool
    def tool1(self, x: int) -> int:
        '''Tool 1 description'''
        return x * 2
    
    @tool
    def tool2(self, y: str) -> str:
        '''Tool 2 description'''
        return y.upper()

# Register toolkit with agent
kit = MyToolKit()
agent.tools = [kit]
```

```python
def register_tools(
    self,
    tools: list,
    task: Optional[Task] = None,
    agent_instance: Optional[Any] = None
```

```python
def remove_tools(
    self,
    tools: Union[Any, List[Any]],
    registered_tools: Dict[str, Any]
) -> tuple[List[str], List[Any]]:
    """Remove tools from the manager.
    
    Handles ALL tool types:
    - Tool names (strings)
    - Function objects
    - Agent objects
    - MCP handlers
    - Class instances (ToolKit or regular classes)
    
    Returns:
        Tuple of (removed_tool_names, removed_original_objects)
    """
```

### Example Usage

```python
from upsonic.tools import ToolManager, tool

# Create manager
manager = ToolManager()

# Define tools
@tool
def add_numbers(a: int, b: int) -> int:
    """Add two numbers together."""
    return a + b

# Register tools
manager.register_tools([add_numbers])

# Execute tool
result = await manager.execute_tool(
    tool_name="add_numbers",
    args={"a": 5, "b": 3}
)

print(result.content)  # 8

# Get all tool definitions
definitions = manager.get_tool_definitions()

# Remove tools
manager.remove_tools(["add_numbers"], registered_tools={})
```

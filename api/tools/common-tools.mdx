## Overview

Upsonic provides a collection of common tools for financial data retrieval, web search, and other utilities. These tools can be easily integrated into your AI agents.

## Financial Tools

#### Example Usage

```python
from upsonic import Agent
from upsonic.tools.common_tools import YFinanceTools

# Create financial tools with specific features
finance_tools = YFinanceTools(
    stock_price=True,
    company_info=True,
    analyst_recommendations=True,
    company_news=True
)

# Or enable all tools
finance_tools = YFinanceTools(enable_all=True)

# Create agent with financial tools
agent = Agent(
    model="openai/gpt-4o",
    tools=finance_tools.functions()
)

# Use the agent
result = agent.run("What is the current stock price of AAPL?")
print(result)
```

## Tool Decorator

**Location:** `src/upsonic/tools/config.py:116`

Decorator to configure tool behavior.

```python
def tool(*args: Any, **kwargs: Any) -> Union[Callable, _ToolDecorator]:
    """Decorator to configure tool behavior.
    
    Use @tool or @tool(...) for configuration.
    """
```

### Example Usage

```python
from upsonic.tools import tool

# Simple tool
@tool
def calculate_sum(a: int, b: int) -> int:
    """Calculate the sum of two numbers."""
    return a + b

# Tool with configuration
@tool(
    requires_confirmation=True,
    cache_results=True,
    timeout=60.0
)
def sensitive_operation(data: str) -> str:
    """Perform a sensitive operation that requires confirmation."""
    return f"Processed: {data}"

# Async tool
@tool
async def fetch_data(url: str) -> dict:
    """Fetch data from a URL."""
    async with aiohttp.ClientSession() as session:
        async with session.get(url) as response:
            return await response.json()
```

## ToolConfig

**Location:** `src/upsonic/tools/config.py:17`

Tool behavior configuration.

```python
class ToolConfig(BaseModel):
    """Tool behavior configuration for the Upsonic framework."""
    
    requires_confirmation: bool = False
    requires_user_input: bool = False
    user_input_fields: List[str] = []
    external_execution: bool = False
    show_result: bool = False
    stop_after_tool_call: bool = False
    sequential: bool = False
    cache_results: bool = False
    cache_dir: Optional[str] = None
    cache_ttl: Optional[int] = None
    tool_hooks: Optional[ToolHooks] = None
    max_retries: Optional[int] = 5
    timeout: Optional[float] = 30.0
    strict: Optional[bool] = None
    docstring_format: str = 'auto'
    require_parameter_descriptions: bool = False
```

### Key Configuration Options

- `requires_confirmation`: Pause and require user confirmation before executing
- `requires_user_input`: Prompt user for input for specified fields
- `external_execution`: Tool execution handled by external process
- `show_result`: Show output to user, NOT sent back to LLM
- `stop_after_tool_call`: Terminate agent run after this tool call
- `sequential`: Require sequential execution (no parallelization)
- `cache_results`: Cache results based on arguments
- `timeout`: Timeout for tool execution in seconds
- `max_retries`: Maximum number of retries allowed

### Example Usage

```python
from upsonic.tools import tool, ToolConfig

config = ToolConfig(
    requires_confirmation=True,
    cache_results=True,
    cache_ttl=3600,  # 1 hour
    timeout=120.0,
    max_retries=3
)

@tool(
    requires_confirmation=True,
    cache_results=True,
    timeout=60.0
)
def my_tool(param: str) -> str:
    """A configured tool."""
    return f"Result: {param}"
```

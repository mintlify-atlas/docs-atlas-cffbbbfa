# Document Processing

Learn how to process complex documents including PDFs with tables, images with OCR, and structured data extraction. Master advanced document handling techniques for RAG systems.

## Document Processing Overview

Upsonic provides powerful document processing capabilities:

- **PDF Processing**: Extract text, tables, and images
- **OCR Support**: Extract text from images and scanned documents
- **Structured Extraction**: Parse tables, forms, and structured data
- **Multi-Format Support**: Handle 20+ document formats
- **Intelligent Loading**: Auto-detect and process any format

## PDF Processing

### Basic PDF Loading

Simple text extraction:

```python
from upsonic.rag.loaders import PDFLoader

pdf_loader = PDFLoader()
documents = pdf_loader.load("document.pdf")

for doc in documents:
    print(f"Page {doc.metadata['page_number']}: {doc.content[:100]}...")
```

Better table extraction using PDFPlumber:

```python

pdf_loader = PDFPlumberLoader(
```

## Image Processing

### Image OCR

Extract text from images:

```python
import pytesseract
from PIL import Image
from upsonic.rag.schemas.data_models import Document

def extract_text_from_image(image_path: str) -> Document:
    """Extract text from image using OCR."""
    image = Image.open(image_path)
    text = pytesseract.image_to_string(image, lang='eng')
    
    return Document(
        content=text,
        metadata={
            "source": image_path,
            "type": "image_ocr",
            "image_size": image.size
        }
    )

doc = extract_text_from_image("screenshot.png")
print(doc.content)
```

### Batch Processing

Process multiple documents efficiently:

```python
from pathlib import Path
from upsonic.rag.loaders.factory import create_intelligent_loaders

# Get all files
docs_dir = Path("./documents")
files = list(docs_dir.glob("**/*"))
files = [f for f in files if f.is_file()]

# Auto-detect loaders
loaders = create_intelligent_loaders(files)

# Process all documents
all_documents = []
for loader, file_path in zip(loaders, files):
    try:
        docs = loader.load(file_path)
        all_documents.extend(docs)
        print(f"✓ Processed {file_path.name}")
    except Exception as e:
        print(f"✗ Failed to process {file_path.name}: {e}")

print(f"\nTotal documents: {len(all_documents)}")
```

## Custom Document Loaders

### Creating Custom Loaders

```python
from upsonic.rag.loaders.base import BaseLoader
from upsonic.rag.schemas.data_models import Document
from typing import List
import json

class CustomJSONLLoader(BaseLoader):
    """Load JSONL (JSON Lines) files."""
    
    def can_load(self, source: str) -> bool:
        """Check if this loader can handle the source."""
        return source.endswith('.jsonl')
    
    def load(self, source: str) -> List[Document]:
        """Load JSONL file into documents."""
        documents = []
        
        with open(source, 'r', encoding='utf-8') as f:
            for line_num, line in enumerate(f, 1):
                try:
                    data = json.loads(line)
                    
                    doc = Document(
                        content=data.get('text', ''),
                        metadata={
                            'source': source,
                            'line_number': line_num,
                            'id': data.get('id'),
                            **data.get('metadata', {})
                        }
                    )
                    documents.append(doc)
                    
                except json.JSONDecodeError as e:
                    print(f"Error on line {line_num}: {e}")
                    continue
        
        return documents

# Use custom loader
loader = CustomJSONLLoader()
documents = loader.load("data.jsonl")
```

### Loader with Preprocessing

```python
class PreprocessingPDFLoader(PDFLoader):
    """PDF loader with text preprocessing."""
    
    def load(self, source: str) -> List[Document]:
        # Load using parent class
        documents = super().load(source)
        
        # Preprocess each document
        for doc in documents:
            # Clean text
            doc.content = self._clean_text(doc.content)
            
            # Add custom metadata
            doc.metadata['word_count'] = len(doc.content.split())
            doc.metadata['char_count'] = len(doc.content)
            doc.metadata['preprocessed'] = True
        
        return documents
    
    def _clean_text(self, text: str) -> str:
        """Clean and normalize text."""
        import re
        
        # Remove multiple spaces
        text = re.sub(r'\s+', ' ', text)
        
        # Remove special characters
        text = re.sub(r'[^\w\s.,!?-]', '', text)
        
        # Normalize whitespace
        text = text.strip()
        
        return text
```

## Best Practices

<Tip>
**Choose the Right PDF Loader**:

- **PDFLoader**: Fast, basic text extraction
- **PDFPlumberLoader**: Better for tables and layouts
- **PyMuPDFLoader**: Best for images and complex formatting
- **DoclingLoader**: Most advanced, slower but comprehensive

```python
# Simple text PDF
loader = PDFLoader()

# PDF with tables
loader = PDFPlumberLoader(extract_tables=True)

# PDF with images
loader = PyMuPDFLoader(extract_images=True)

# Complex documents
loader = DoclingLoader(detect_layout=True)
```
</Tip>

<Warning>
**OCR Performance**: OCR is slow and resource-intensive:

- Use only when necessary (scanned documents)
- Process in batches
- Consider caching results
- Use appropriate OCR quality settings

```python
# Fast OCR (lower quality)
pdf_loader = PDFPlumberLoader(
    use_ocr=True,
    ocr_dpi=150  # Lower DPI = faster
)

# Better OCR (slower)
pdf_loader = PDFPlumberLoader(
    use_ocr=True,
    ocr_dpi=300  # Higher DPI = better quality
)
```
</Warning>

<Tip>
**Metadata Strategy**: Add useful metadata for filtering:

```python
class MetadataEnhancedLoader(PDFLoader):
    def load(self, source: str) -> List[Document]:
        documents = super().load(source)
        
        for doc in documents:
            # Add custom metadata
            doc.metadata['document_type'] = self._detect_type(doc)
            doc.metadata['language'] = self._detect_language(doc)
            doc.metadata['reading_time'] = len(doc.content.split()) / 200
        
        return documents
```
</Tip>

<Tip>
**Error Handling**: Always handle loading errors:

```python
for file_path in files:
    try:
        loader = get_loader_for_file(file_path)
        documents = loader.load(file_path)
        process_documents(documents)
    except FileNotFoundError:
        print(f"File not found: {file_path}")
    except PermissionError:
        print(f"Permission denied: {file_path}")
    except Exception as e:
        print(f"Error loading {file_path}: {e}")
        continue
```
</Tip>

<Note>
**Memory Management**: Large documents can use significant memory:

```python
# Process in chunks for large files
def process_large_pdf(file_path: str, chunk_size: int = 10):
    loader = PDFLoader()
    
    # Process pages in batches
    total_pages = get_pdf_page_count(file_path)
    
    for start_page in range(0, total_pages, chunk_size):
        end_page = min(start_page + chunk_size, total_pages)
        
        # Process chunk
        documents = loader.load(
            file_path,
            pages=list(range(start_page, end_page))
        )
        
        # Process immediately and clear
        yield from documents
        del documents
```
</Note>
